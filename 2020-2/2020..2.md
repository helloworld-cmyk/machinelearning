Câu 4: Cho CSDL gồm các quan hệ:
Kháchhàng(MãKH, Họtên, Sốđiệnthoại, Cơquan)
Nhàchothuê(MãN, Địachỉ, Giáthuê, Tênchủnhà)
Hợpđồng(MãN, MãKH, Ngàybắtđầu, Ngàykếtthúc)  
Các lệnh tạo các bảng trên như sau:
SQL
CREATE TABLE Kháchhàng(
MãKH char(8) primary key,
Họtên char(30) NOT NULL,
Sốđiệnthoại char(11),
Cơquan char(30);
);

CREATE TABLE Nhàchothuê(
MãN char(6) primary key,
Địachỉ varchar(70),
Giáthuê int,
Tênchủnhà char(30),
);

CREATE TABLE Hợpđồng(
MãN char(6) primary key,
MãKH char(8) primary key,
Ngàybắtđầu date,
Ngàykếtthúc date,
foreign key references Nhàchothuê(MãN),
foreign key references Kháchhàng(MãKH)
);
Hỏi lệnh tạo bảng nào có lỗi? (Tham chiếu hình ảnh: image_0fc6c5.jpg)
A. Lệnh tạo bảng Kháchhàng và bảng Hợpđồng
B. Cả 3 lệnh tạo bảng trên đều lỗi
C. Lệnh tạo bảng Nhàchothuê và bảng Hợpđồng
D. Chỉ có lệnh tạo bảng Hợpđồng lỗi
ans: B [suy luận]
