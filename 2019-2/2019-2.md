DuAn(DA, TenDA, KinhPhi)

Câu lệnh SQL nào sau đây đưa ra HoTen nhân viên và Số dự án họ tham gia
Chọn 1 đáp án:
A. SELECT HoTen, SUM(DA) FROM NhanVien, ThamGia WHERE NhanVien.NV = ThamGia.NV GROUP BY NhanVien.NV
B. SELECT HoTen, COUNT(DA) FROM NhanVien, ThamGia WHERE NhanVien.NV = ThamGia.NV GROUP BY NhanVien.NV
C. SELECT HoTen, COUNT(DA) FROM NhanVien, ThamGia WHERE NhanVien.NV = ThamGia.NV GROUP BY NV
D. SELECT HoTen, COUNT(NV) FROM NhanVien, ThamGia WHERE NhanVien.NV = ThamGia.NV GROUPBY NhanVien.DA
ans: B [suy luận]

Câu 5: Cho CSDL gồm các quan hệ:
Kháchhàng(MãKH, Họtên, Sốđiệnthoại, Cơquan)
Nhàchothuê(MãN, Địachỉ, Giáthuê, Tênchủnhà)
Hợpđồng(MãN, MãKH, Ngàybắtđầu, Ngàykếtthúc)  
Các lệnh tạo các bảng trên như sau:
SQL
CREATE TABLE Kháchhàng(
MãKH char(8) primary key,
Họtên char(30) NOT NULL,
Sốđiệnthoại char(11),
Cơquan char(30)
);

CREATE TABLE Nhàchothuê(
MãN char(6) primary key,
Địachỉ varchar(70),
Giáthuê int,
Tênchủnhà char(30)
);

CREATE TABLE Hợpđồng(
MãN char(6) primary key ,
MãKH char(8) primary key,
Ngàybắtđầu date,
Ngàykếtthúc date,
foreign key references Nhàchothuê(MãN),
foreign key references Kháchhàng(MãKH)
);
Hỏi lệnh tạo bảng nào có lỗi?
Chọn 1 đáp án:
A. Cả 3 lệnh tạo 3 bảng đều có lỗi
B. Lệnh tạo bảng Kháchhàng và bảng Hợpđồng
C. Lệnh tạo bảng Hợpđồng
D. Lệnh tạo bảng Nhàchothuê
ans: C [suy luận]
